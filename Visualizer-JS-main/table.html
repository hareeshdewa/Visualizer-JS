Module: Sorting Visualizer

Purpose The Sorting Visualizer module is designed to provide an interactive web-based tool for visualizing various sorting algorithms. Its primary purpose is to help users understand how different sorting algorithms work by visually demonstrating the sorting process step by step. Users can interact with the visualizer to see how algorithms such as Quick Sort, Selection Sort, Bubble Sort, Insertion Sort, and Merge Sort operate on arrays of integers. Additionally, the module allows users to adjust the size of the input array, select different sorting algorithms, and observe the time and memory complexity of each algorithm.
Components
  • Sorting Algorithms:
    • Quick Sort
    • Selection Sort
    • Bubble Sort
    • Insertion Sort
    • Merge Sort
  • Array Visualization: The module provides a graphical representation of the input array, with bars representing individual elements. During sorting, the bars move to visualize the comparison and swapping steps of the sorting algorithms.
  • User Interaction:
    • Restart Button: Allows users to reset the array to its initial state for sorting.
    • Array Size Adjustment: Users can dynamically change the size of the input array to observe sorting with different input sizes.
    • Algorithm Selection: Users can select different sorting algorithms to visualize their sorting process.
  • Time and Memory Complexity Display: The module displays the time and memory complexity of each sorting algorithm in a table format. Users can compare the efficiency of different algorithms based on their time and space requirements.
